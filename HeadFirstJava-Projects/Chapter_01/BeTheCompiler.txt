**This program will run infinit loop as x is always less than 10, to fix it, we to increase x by one on each loop
FIXED:

class Exercise1a {
	public static void main(String[] args) {
	int x = 1;
	while (x < 10) {
		x = x + 1;
		if (x > 3) {
		System.out.println("big x");
		}
	}	
     }	
}
